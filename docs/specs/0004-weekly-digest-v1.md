# Спека 0004 — Weekly digest v1 (DRAFT)

Статус: **draft**

## Факты функционала

- Система генерирует weekly digest по активному проекту.
- Digest отвечает: что произошло, что обещано, где риски, что делать дальше.
- Digest должен ссылаться на evidence.

## Acceptance criteria

- Digest не включает данные других проектов.
- Digest содержит ссылки на первоисточники.

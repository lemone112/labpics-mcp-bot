# Спека 0002 — Commitments v1 (DRAFT)

Статус: **draft**

## Факты функционала

- Система извлекает commitments (договорённости / обязательства / следующие шаги) из переписки.
- Commitments строго project-scoped и evidence-backed.

## Non-goals (v1)

- автоматическая отправка внешних сообщений без approvals

## Acceptance criteria

- Каждый commitment содержит ссылку на исходные сообщения (evidence).
- Нет commitments без project_id.
